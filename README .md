# 🏫 Portal Gestão Escolar v3.0

Sistema profissional para gestão de laboratórios, monitoria e calendário escolar.

## ✨ Funcionalidades

### 🔐 Autenticação
- Login com email e senha
- Cadastro de novos usuários
- Sessão persistente por 8 horas
- Usuário admin padrão

### 📅 Calendário
- Visualização mensal
- Marcação de feriados e eventos
- Clique direito para personalizar
- Dias bloqueados configuráveis

### 🧪 Laboratórios
- 3 laboratórios: LEI, Hardware, Multimídia
- Reserva por horário
- Seleção de professores e turmas

### 🛡️ Monitoria
- 5 postos de trabalho fixos
- Escala por data
- Atribuição de monitores e turmas

### ⚙️ Configurações
- Cadastro de professores
- Cadastro de turmas
- Cadastro de monitores
- Configuração de dias bloqueados

## 🚀 Como Usar

### Acesso Inicial
- **Email:** admin@escola.com
- **Senha:** admin123

### Navegação
1. Menu principal com 4 cards
2. Clique em qualquer laboratório para reservas
3. "Monitoria" para escala dos postos
4. "Configurar" para cadastros e configurações

### Calendário
- Clique em um dia para selecionar
- Clique direito para marcar feriado/evento
- Dias bloqueados (configuráveis) aparecem em vermelho

### Reservas
- Selecione o dia no calendário
- Escolha professor e turma para cada horário
- Dias bloqueados não permitem reservas

## 🛠️ Tecnologias

- HTML5
- CSS3 (com variáveis CSS)
- JavaScript (modular)
- LocalStorage para persistência
- Design responsivo
